# 🎓 Student_Management

## 📌 Overview

Student_Management is a simple web-based application built using PHP, MySQL, HTML, CSS, and JavaScript to manage student records.

## 🚀 Features

* ➕ Add student
* 📋 View students
* ✏️ Update student details
* ❌ Delete student records
* 🔐 Login system (admin/user)

## 🛠️ Technologies Used

* 💻 PHP
* 🗄️ MySQL
* 🌐 HTML
* 🎨 CSS
* ⚡ JavaScript

## 🔒 Security (Hashing & Sanitizing)

* 🔐 Passwords are securely stored using hashing (e.g., `password_hash()` in PHP)
* 🛡️ User inputs are sanitized to prevent SQL Injection and XSS attacks
* ✅ Use of prepared statements for safe database queries

## ⚙️ Setup

1. Install XAMPP
2. Start Apache and MySQL
3. Create a database named Student_Management
4. Place the project folder in `htdocs`
5. Run the project:
   [http://localhost/Student_Management/](http://localhost/Student_Management/)
